#  Retail Sales & Analytics Dashboard

---

## Project Overview
This project focuses on building an end-to-end data analytics solution for a retail business. The goal is to analyze sales, profit, customer behavior, regional performance, and logistics efficiency to generate actionable business insights.

The dashboard helps stakeholders understand performance trends, identify inefficiencies, and make data-driven decisions.

---

## Business Objectives
- Identify sales trends over time  
- Calculate key KPIs (Sales, Profit, Average Order Value)  
- Analyze product and category performance  
- Evaluate customer contribution and segmentation  
- Compare regional performance  
- Improve operational and logistics efficiency

##  Dataset Description
21 Columns and 9994 Rows
The dataset includes:
- Sales, Profit, Quantity, Discount

- Customer Segment  
- Region,State & City  
- Order Date & Shipping Date  

Time Period: 2014–2017

## Key Insights
- Sales growth is volume-driven, not profit-driven  
- High discounts reduce profitability  
- Profit is concentrated in a few products  
- Some customers and regions generate losses  
- Logistics inefficiencies cause delivery delays  

---

## Recommendations
- Control discounting to improve margins  
- Focus on high-profit products  
- Improve customer segmentation  
- Optimize underperforming regions  
- Enhance logistics efficiency
